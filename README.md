# String-Operation-in-C-Programming-Language
This project offers a menu-driven C program that does manual string manipulation without using any library functions. It offers functionality for case conversion, reversing strings, and counting character frequencies. This project is ideal for BCA Semester 1 assignments.
String Operation Project (C Programming)
A menu-driven C application designed to perform various string manipulations without relying on standard library functions like strupr(), strrev(), or tolower(). This project demonstrates a deep understanding of ASCII values, pointer/array traversal, and modular programming.


üöÄ Features
The program provides an interactive menu with the following manual operations:

Case Conversion: Toggle between full Uppercase and Lowercase.

String Reversal: Reverse the entire string or reverse each word while maintaining its original position.

Word Formatting: Automatically capitalize the first or last letter of every word in a sentence.

Character Analytics: Count the occurrence of every individual character within the input string.


üõ†Ô∏è Implementation Details
No String Functions: All logic is written using manual loops to handle null-terminators (\0) and character offsets.

Modular Design: Each operation is contained within its own function for better readability and maintenance.

Interactive Menu: Built-in functionality to continue with the same string or start fresh with new input.


üìñ How to Run
1. Ensure you have a C compiler installed (like GCC or TurboC).

2. Compile the file:
gcc stringoperation.c -o stringop

3. Run the executable:
./stringop
